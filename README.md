# TODO APP :->
* A simple Todo Web App made using Express
* It is a simple TODO application using NodeJS and MongDB. where a user of this application can create,edit and delete all the ToDo items 
# How to use :->
* Get NodeJs on your System.
* Clone this repository .
* Open in terminal/cmd
* Do npm install
* Go to Web Browser and hit https://localhost:8000
